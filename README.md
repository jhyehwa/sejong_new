## 주요기능
- 회원가입/로그인
- 메뉴(등록/수정/삭제)
- 예약
- 공지/이벤트
- 자유게시판
- QnA
<br/>

## 특징
- Enterprise 환경에서의 자바 기반 웹 프로그래밍 개발 기술 습득
- WAS, JSP 및 Servlet  사용
- 필터, 서블릿 이벤트, JSTL, 커스텀 태그 사용
- Database Connection Pool을 이용 Connection 재 사용 및 처리 시간 단축
- MVC2 패턴을 활용한 프로젝트
<br/>
